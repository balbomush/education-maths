# Education Maths — тренажёр задач по математике

Education Maths — это простое офлайн-приложение для Windows, в котором ученик 5–9 классов решает задачи по математике, вводит ответы и сразу получает проверку и короткую обратную связь.

- **Диапазон классов**: 5–9.
- **Темы**: арифметика, дроби, проценты, уравнения, пропорции, геометрия и др.
- **Типы задач**: числовой ответ, выбор вариантов (multiple choice), простые текстовые задачи.
- **Банк задач**: для каждой темы внутри каждого класса сгенерировано по несколько сотен параметрических задач.

## Как работает программа

1. **Главное окно**
   - выбираешь класс (5–9);
   - выбираешь тему;
   - при желании задаёшь сложность (1–3) и количество задач за сессию (5, 10, 20 или все).

2. **Режим решения задач**
   - программа по очереди показывает задачи выбранной темы и класса;
   - для числовых/текстовых задач вводишь ответ в поле ввода;
   - для задач с выбором ответа отмечаешь один или несколько вариантов;
   - по кнопке «Проверить» отображается результат («Верно» / подсказка про ошибку);
   - после нескольких неверных попыток можно увидеть более подробный разбор (если он есть).

3. **Прогресс**
   - для каждой задачи сохраняется количество попыток и число верных ответов;
   - на экране прогресса показывается статистика по классам и темам (количество попыток и процент верных);
   - из окна прогресса можно сохранить статистику в CSV‑файл для последующего анализа.

4. **Адаптивность**
   - если сложность не выбрана явно, программа подбирает задачи так, чтобы постепенно усложнять тему при высоком проценте верных ответов и упрощать при низком.

## Как запустить на Windows

Самый простой способ — использовать уже собранный `.exe`‑файл, Python ставить не нужно.

- **Скачать**: [education-maths.exe](https://github.com/balbomush/education-maths/raw/main/dist/education-maths.exe)
- Сохранить файл на компьютер и запустить двойным кликом. При первом запуске Windows может спросить подтверждение запуска файла, скачанного из интернета.
